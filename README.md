Inty's Exherbo repo
===================

Exheres packages for luakit, rtm, moreutils, dictd, dict, gcal, boxes etc.

/etc/paludis/repositories/inty.conf:

    format = e
    location = /var/db/paludis/repositories/inty
    sync = git://github.com/inty/exherbo.git
